VUE项目搭建总结

1、安装 node

  • 1、 Windows 安装包(.msi)
    • ( 1 ) 32 位安装包下载地址 : https://nodejs.org/dist/v4.4.3/node-v4.4.3-x86.msi
    • ( 2 ) 64 位安装包下载地址 : https://nodejs.org/dist/v4.4.3/node-v4.4.3-x64.msi
    • 下载完安装一直下一步即可
  • 2、 Windows 二进制文件 (.exe)安装
    • （ 1 ）32 位安装包下载地址 : http://nodejs.org/dist/v0.10.26/node.exe
    • （ 2 ）64 位安装包下载地址 : http://nodejs.org/dist/v0.10.26/x64/node.exe
    • 点击 Run（运行）按钮将出现命令行窗口；进入 node.exe 所在的目录，输入node -v 查看安装版本，出来node版本即成功。

2、安装 npm

NPM是随同NodeJS一起安装的包管理工具，能解决NodeJS代码部署上的很多问题，由于新版的nodejs已经集成了npm，所以之前npm也一并安装好了。同样可以通过输 入 "npm -v" 来测试是否成功安装

npm升级方法：
  1. npm install npm -g( windows版)
  2. sudo npm install npm -g（非windows版）

3、安装VUE（npm 版本需要大于 3.0，如果低于此版本需要升级它）

  • npm install vue -g
  • npm install vue-cli -g
  • npm install webpack -g ( WebPack可以看做是模块打包机 )

4、创建VUE项目

( 1 ) vue项目创建

  • 在cmd命令窗口中输入：vue init webpack demo
  • 按照提示自动打开浏览器出现VUE图标即完成项目创建

( 2 ) vue项目依赖第三方包

  1. 安装vvue-router: npm install vue-router --save
  2. 安装less和less-loader npm install less less-loader --save
    1. 配置less路径:
    2. 在build---> webpack.base.conf.js 添加 { test: /\.less$/,loader: 'style-loader!css-loader!less-loader'}
  4. 安装sass:
    1. ---->npm install node-sass --save
    2. ---->npm install sass-loader --save
    3. 通常使用npm安装会出现以下报错，安装失败。（网路问题）

    4. 可以通过淘宝的npm镜像安装node-sass，解决以上问题。
  6. 淘宝镜像安装：
    1. -->npm install -g cnpm --registry=https://registry.npm.taobao.org （安装淘宝镜像）
    2. 使用：-->cnpm install node-sass --save （使用淘宝镜像安装node-sass）
  8. 图片处理（ element-ui )：（一种解决跨域问题，也可以用node进行转换）
    1. 在package.json 的 dependencies 字段下面增加"element-ui":"^1.3.4" 这里的1.3.4是需要安装的那个版本
      1. 在main.js 下面增加element-ui 的应用
      2. import ElementUI from 'element-ui'
      3. import 'element-ui/lib/theme-default/index.css'
      4. Vue.use(ElementUI)
    3. 应用（例）：<el-button type="primary">主要按钮</el-button>
  10. 安装vuex （必须显式地通过 Vue.use()来安装 Vuex ）
    1. Vuex 是一个专为 Vue.js 应用程序开发的状态管理模式,这个状态自管理应用包含以下几个部分：
    2. state，驱动应用的数据源；
    3. view，以声明方式将 state 映射到视图；
    4. actions，响应在 view 上的用户输入导致的状态变化
    5. 需要构建是一个中大型单页应用，您很可能会考虑如何更好地在组件外部管 理状态，Vuex 将会成为自然而然的选择
  12. 安装 axios -->npm install axios -s
    1. 在 main.js 中引入 axios-->import axios from 'axios'
    2. 这时候如果在其它的组件中，是无法使用 axios 命令的。但可以将 axios 改写为 Vue 的原型属性，Vue.prototype.$ajax = axios
    3. 在 main.js 中添加了这两行代码之后，就能直接在组件的 methods 中 使用 $ajax 命令
